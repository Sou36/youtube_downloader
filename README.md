～ファイルの説明～

youtube.py → Youtube上の動画を保存したい際に使用します。



～実行環境～

pythonのyt-dlpというライブラリと、ffmpegを使用しているのでそれぞれ準備してから実行してください。

yt-dlpはCMDで
```
pip install yt-dlp
```

ffmpegはCMDで
```
winget install --id=Gyan.FFmpeg -e
```

すれば簡単に入れれます。過去に入れたことがあればしなくていいです。
